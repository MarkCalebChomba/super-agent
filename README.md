# Internet Smart Agent

Autonomous multi-agent system that makes money online across 12+ income verticals. Self-modifying — SuperAgent creates, edits, and stops agents autonomously (up to 256).

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `OPENROUTER_API_KEY` | Yes | OpenRouter API key (free tier) |
| `GROQ_API_KEY` | Yes | Groq API key (free tier) |
| `TELEGRAM_BOT_TOKEN` | Yes | Bot token from @BotFather |
| `TELEGRAM_CHAT_ID` | Yes | Your Telegram user ID |
