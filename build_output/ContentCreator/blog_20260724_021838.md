**1. Automating File Management with Python**

Python's `os` and `shutil` libraries make it easy to automate file management tasks, such as renaming, copying, and moving files. For instance, you can create a script to rename all files in a folder with a specific prefix or suffix. Here's an example code snippet:

```python
import os

# Define the folder path and prefix
folder_path = '/path/to/folder'
prefix = 'prefix_'

# Loop through all files in the folder
for file in os.listdir(folder_path):
    # Rename the file with the prefix
    os.rename(os.path.join(folder_path, file), os.path.join(folder_path, prefix + file))
```

**2. Automating Data Extraction and Analysis with Python**

Python's `pandas` library is a data scientist's best friend, offering a powerful tool for data manipulation and analysis. You can use `pandas` to extract data from various sources, such as Excel files, CSV files, or databases. For example, you can create a script to extract specific columns from a CSV file and perform calculations on the data:

```python
import pandas as pd

# Load the CSV file
df = pd.read_csv('data.csv')

# Extract specific columns
columns = ['Column1', 'Column2']
data = df[columns]

# Perform calculations
result = data['Column1'] + data['Column2']
```

**3. Automating Email and Notification Tasks with Python**

Python's `smtplib` and `email` libraries make it easy to automate email tasks, such as sending automated reports or notifications. You can create a script to send a daily or weekly report to stakeholders, or set up a notification system to alert team members of important events. Here's an example code snippet:

```python
import smtplib
from email.mime.text import MIMEText

# Define the email settings
subject = 'Daily Report'
body = 'This is the daily report'
from_addr = 'your_email@example.com'
to_addr = 'recipient_email@example.com'

# Create the email message
msg = MIMEText(body)
msg['Subject'] = subject
msg['From'] = from_addr
msg['To'] = to_addr

# Send the email
server = smtplib.SMTP('smtp.example.com')
server.sendmail(from_addr, to_addr, msg.as_string())
server.quit()
```
